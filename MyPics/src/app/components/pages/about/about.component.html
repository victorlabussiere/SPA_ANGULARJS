Sobre o MyPics!

Compartilhe suas fotos e receba comentários das pessoas que mais gosta!

O MyPics é um projeto realizado para estudos de boas práticas e técnicas mais utilizadas com AngularJS.

Este projeto foi inspirado no Curso Gratuito do Mathues Batisti, disponibilizado em seu canal: youtube.com/@matheusbattisti

Para o backend, foi utilizado o framework AdonisJS e nele foram implementados todos os serviços para o CRUD com as informações do cliente.

O principal objetivo deste projeto é externar minhas habilidades como desenvolvedor frontend Javascript, com Angular.
